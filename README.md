# Interactive Online Store Sales Dashboard

Project Description:
I built a clean and interactive sales analysis dashboard designed to turn raw e-commerce data into clear, actionable insights. This project focuses on helping businesses quickly understand performance across categories, products, and pricing strategies.

The dashboard highlights key metrics like total revenue, number of orders, average discount, and customer reviews—giving a quick snapshot of overall performance. It also breaks things down further with visuals showing revenue by category and sub-category, top-performing products, and how pricing (actual vs discounted) impacts sales.

To make it more user-friendly, I included filters for category, price range, and product selection so users can explore the data from different angles without needing technical skills. The layout is simple, visually consistent, and built for easy decision-making.

## Key Features:

Dynamic KPI cards (Revenue, Orders, Discounts, Reviews)

Revenue breakdown by category and sub-category

Sales performance of top products

Pricing analysis (actual vs discounted trends)

Category-based rating insights

Interactive filters for deeper exploration

This project reflects my approach to data analysis: not just presenting numbers, but telling a clear story that helps businesses make smarter decisions.
